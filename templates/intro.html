quick answers to questions you may have

why do I need to log in? logging into your tumblr account allows the app to see who follows you. you need to sign in because this information is not public
what can this application do? this application will request read-only permissions for your tumblr account. it will not make any changes to your account
does this use cookies? this application uses an encrypted session cookie in order to store your tumblr sign in state, and nothing else. no information about you is retained on the server. by using this application you're saying that you're okay with this cookie.
can I see the code? yes! absolutely. the code can be found on github here. please feel free to copy this or contribute in whatever way you'd like.
what was this made with? this application was written in python3, using the flask web framework. in addition to flask and its dependencies, it uses requests to handle http requests server-side
